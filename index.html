Componentes del sitio web o página web

Header

Contiene un encabezado donde se permite navegar a travez de las páginas

Main

Tiene un contenido main, donde es el contenido principal de la página

Footer

Contiene una parte final, osea un footer donde se ubica al final de la página

Semantico

El sitio web y sus páginas contiene Semantica Ej:section, article, main, nav, header, footer, aside

Grid

Usamos grid que permite mover bloques a travez de los ejes Y, X, display:grid;

Flexbox

Usamos flexbox que permite mover bloques pero solo a travez de un eje, sea Y o X, solo se puede eligir uno, display:flex;

Svg

Utilizamos componentes Svg, que permite una mayor optimización a la página web

TypeScript

Utilizamos TypeScript lo que nos permitio reutilizar mas los componentes con mayor facilidad

Astro

Usamos el FrameWork Astro, lo que nos permitio completar el sitio web de una forma mas productiva y efectivda

Preguntas que nos hicimos

¿Y si mejor hacemos un video o otra cosa?

Nos planteamos la pregunta de mejor dejar la página web y hacer otra presentación mas facil, pero no al final decidimos no rendirnos por una dificultad que se puede superar con disciplina

¿Mandamos a hacer la página?

Al vernos acorrolados por la primera pregunta, se nos vino por la mente la idea de mandar a hacer la página, pero seguimos firmes en nuestra decisión y no

¿Copiamos el código o usamos una plantilla?

Al no usar las 2 preguntas anteriores podríamos perfectamente copiar el código de otra página o usar una plantilla, pero no logramos descartar esas ideas absurdas y crear nuestra propia página

¿Mayor desafío?

Nuestro mayor desafío fue el choque de ídeas cada quien quería hacer otra cosa y optmos por dividir roles

Página inspirada en AstroWindy en AstroShip

Se realizo gracias a la estructura, cabe recalcar que nunca usamos el código de astrowind, astroship, que son plantillas, lo que realmente hicimos fue ver su estructura y aprendes a hacer desde 0 todo lo que ellos hicieron,nunca usamos su código o su plantilla nosotros creamos desde 0 todo
