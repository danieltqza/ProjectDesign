Componentes del sitio web o página web

Header

Contiene un encabezado donde se permite navegar a travez de las páginas

Main

Tiene un contenido main, donde es el contenido principal de la página

Footer

Contiene una parte final, osea un footer donde se ubica al final de la página

Semantico

El sitio web y sus páginas contiene Semantica Ej:section, article, main, nav, header, footer, aside

Grid

Usamos grid que permite mover bloques a travez de los ejes Y, X, display:grid;

Flexbox

Usamos flexbox que permite mover bloques pero solo a travez de un eje, sea Y o X, solo se puede eligir uno, display:flex;

Svg

Utilizamos componentes Svg, que permite una mayor optimización a la página web

TypeScript

Utilizamos TypeScript lo que nos permitio reutilizar mas los componentes con mayor facilidad

Astro

Usamos el FrameWork Astro, lo que nos permitio completar el sitio web de una forma mas productiva y efectivda
